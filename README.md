# Minipuyo

Entry for JS13K 2023.

A super puyo puyo remake with a fancy medieval wall as a background.

![screenshot](https://github.com/gogoprog/minipuyo/raw/master/res/minipuyo.png)

# How to play

* Moving : Arrows or WASD
* Rotating : O or P
* Validating : Enter

# Tools

 * Haxe Compiler 4.3.1
 * Terser
 * Gimp
